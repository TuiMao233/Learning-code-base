## Vuex组件中使用方式

#### 在任意组件中显示state数据内容

```javascript
/* 状态数据(state.js) */
state = { cliNum: 0 }
```

```html
<!-- 任意组件 -->
<div>{{ $store.state.cliNum }}</div>
```

#### 在任意组件内使用计算值

```javascript
/* 状态数据(getters.js) */
getters = {
  	type (state) {
   		return (state.cliNum % 2) === 0 ? '偶数' : '奇数'
  	}
}
```

```html
<!-- 任意组件 -->
<div>{{ $store.getters.type }}</div>
```

#### 在任意组件使用改变state中属性的函数

1. **传入单一形参**

   ```javascript
   /* 状态数据(actions.js,mutations.js) */
   actions = {
     	qaq ({commit}, Arg) { commit('QAQ'), Arg }
   }
   mutations = {
     	QAQ (state, Arg) { console.log('我运行啦' + Arg) }
   }
   ```

   ```html
   <!-- 任意组件 -->
   <button @click="$store.dispatch('qaq(Arg)')">+</button>
   ```

2. **数组方式传入多个形参**

   ```javascript
   /* 状态数据(actions.js,mutations.js) */
   actions = {
     	qaq ({commit}, Args) { commit('QAQ'), {Arg1:Args[0], Arg2:Args[0]} }
   }
   mutations = {
     	QAQ (state, {Arg1, Arg2}) { console.log(Arg1,Arg2) } 
   }
   ```

   ```javascript
   /* 任意组件 */
   export default {
   	mounted () {
   		this.$store.dispatch('qaq', ['wd','nmd']) // 控制台输出'wdnmd'
   	}
   }
   ```

3. **数组方式传入多个形参（AcMutations框架写法）**

   ```javascript
   /* 状态数据(acMutions.js) */
   qaq (state, {arg1, arg2}) {
       console.log(Arg1,Arg2)
   }
   ```

   ```javascript
   /* 任意组件 */
   export default {
   	mounted () {
   		this.$store.dispatch('qaq', ['wd','nmd']) // 控制台输出'wdnmd'
   	}
   }
   ```

#### 在任意组件定义vuex的属性和方法

- **组件定义vuex的state属性**

  ~~~javascript
  /* 定义 */
  import {mapState} from 'vuex'
  export default {
      computed: { ...mapState(['axo']) }
  }
  ~~~

  ~~~html
  <!-- 组件使用 -->
  <p>{{axo}}</p>
  ~~~

- **组件定义vuex的getters计算属性**

  ~~~javascript
  /* 定义 */
  import {mapGetters} from 'vuex'
  export default {
      computed: { ...mapGetters(['QAQ']) }
  }
  ~~~

  ~~~html
  <!-- 组件使用 -->
  <p>{{QAQ}}</p>
  ~~~

- **组件定义vuex的actions方法**

  ~~~javascript
  /* 定义 */
  import {mapState} from 'vuex'
  export default {
  	methods: { ...mapActions(['QAq']) }
  }
  ~~~

  ~~~html
  <!-- 组件使用 -->
  <button @click="QAq(['wd','nmd'])">点我一下</button>
  ~~~
